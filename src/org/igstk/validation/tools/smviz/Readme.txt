IGSTK Validation tools - SMVIZ (State Machine Visualization).

Steps to run the SMVIZ, are as follows:

	1.	Check out the IGSTK Validation Tools Code.
	2. 	$ant run-smviz

	To generate the distributables only for simulator
	1. 	$ant dist-smviz

The required softwares and versions are:

	1.	JDK 1.5 and above.
	2. 	Ant 1.7 and above.

IGSTK Validation tools - Simulator.

Steps to run the simulator, without the SMVIZ GUI are as follows:

	1.	Check out the IGSTK Validation Tools Code.
	2. 	$ant run-simulator

	To generate the distributables only for simulator
	1. 	$ant dist-simulator

The required softwares and versions are:

	1.	JDK 1.5 and above.
	2. 	Ant 1.7 and above.
